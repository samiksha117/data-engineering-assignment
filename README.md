# assignment-solutions



## Overview
This repository contains my solutions for the SQL, Spreadsheet, and Python tasks given in the assignment.



## Folder Structure

- SQL/  
  Contains queries for both Hotel Management and Clinic Management problems.

- Spreadsheets/  
  Contains the Excel file used for ticket analysis.

- Python/  
  Contains Python scripts for the given problems.



## Approach

### SQL
I solved the queries using joins, grouping, and aggregation.  
For ranking-related problems, I used window functions like RANK and DENSE_RANK.  
I also used CTEs to make queries more readable.



### Spreadsheet
I used lookup functions to map data between sheets.  
To check same day and same hour conditions, I created helper columns.  
Then I used pivot tables to get outlet-wise counts.

---

### Python
I used simple logic and loops:
- Converted minutes into hours and minutes format
- Removed duplicate characters from a string



## Assumptions
- Datetime values are in standard format  
- Billing amount is calculated using quantity and rate  
- Monthly calculations are based on given data
